My Personal Health Record

Inactive Features of the Portal Index Page

Header Cards

The "Messages," "Appointments," "Prescriptions," and "Case Management" header cards will alert the user to the number of items that need their attention.

For example, the "Prescriptions" card will alert the user to the following:

  • The number of prescriptions that will need renewal; or
  • The number of prescriptions that will need to be refilled; or
  • The number of new prescriptions that have not been reviewed.

Updated Vitals

The "Updated Vitals" section in the final version utilizes a PHP script to pull the most current vitals from the database. These vitals are entered in the vital portal. Since I am using my own information to test these functions, I decided not to include a live version into my capstone project.

Care Team

The "Care Team" section in the final version utilizes a PHP script to pull the most current members of the patient's care team from the database.

Recent After Visits

The "Recent After Visits" section in the final version utilizes a PHP script to pull the three most current After Visit Reports from the database. The After Visit report is an important asset to the patient as it serves as the reminder of the results of the appointment/visit.
